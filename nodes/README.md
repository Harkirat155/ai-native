# Ecosystem nodes (starter)

This directory contains thin adapters to plug the Bridge into popular agent orchestration frameworks.

Initial goal: provide **simple, stable wrappers** around:
- the Core 15 transactional methods
- event subscriptions (`events.subscribe`)
- LLM toolpack (`docs/protocol-v1.json`)
